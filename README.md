Starte mit

docker compose up -d

MinIO-Buckets erreichbar über:

http://localhost:9001 (staging)

http://localhost:9002 (images)

http://localhost:9003 (videos)

MariaDB läuft unter Port 3306, Login via root und Passwort aus .env.
